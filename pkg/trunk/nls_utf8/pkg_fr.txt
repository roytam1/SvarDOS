#### Help ####

1.0:PKG est le gestionnaire de paquets de SvarDOS.
1.19:Usage:
1.20:pkg install <package[.svp]>   installe le paquet
1.21:pkg update <package[.svp]>    met à jour (ou réinstalle) le paquet
1.22:pkg rm <package>              enlève le paquet
1.23:pkg files <package>           liste les fichiers appartenant au paquet
1.24:pkg list [<filter>]           liste les paquets installés
1.25:pkg check [<package>]         vérifie l'intégrité des fichiers exécutables
1.26:pkg check+ [<package>]        comme ci-dessus mais pour TOUS installés
1.27:pkg unzip <file.zip>          ouvre une archive ZIP
1.28:pkg crc32 <file>              calcule la valeur CRC32 d'un fichier
1.29:pkg ziplist <file.zip>        liste les fichiers au sein de l'archive ZIP
1.40:PKG est publié sous licence MIT.
1.41:Cet outil est configurable via %DOSDIR%\\PKG.CFG
