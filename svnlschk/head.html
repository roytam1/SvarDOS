Svarog386 NLS support

NLS stands for "National Language Support"

Svarog386 can be installed in several languages. This page lists all languages available today, along with simple statistics about the translation status of each core package.

Your language is either missing or incomplete, and you'd like to contribute to the Svarog386 project by providing translations in your native language? Great! Simply download THIS FILE, it contains all translation files shipped with Svarog386, so you should be up to speed in no time.
